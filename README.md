### Presupuesto App JS
#### Una aplicación web interactiva diseñada para la gestión financiera personal, permitiendo a los usuarios establecer un presupuesto #### base, registrar gastos detallados y visualizar su saldo disponible en tiempo real.

### Características
#### Gestión de Presupuesto: Permite ingresar y acumular un presupuesto inicial.

#### Control de Gastos: Registro de gastos con nombre y monto.

#### Validaciones Inteligentes: El sistema impide registrar gastos si no hay presupuesto suficiente o si los campos están vacíos.

#### Interfaz Dinámica: Actualización automática de la tabla de gastos y saldos mediante manipulación del DOM.

#### Eliminación de Registros: Posibilidad de borrar gastos específicos, recalculando el saldo de forma inmediata.

### Tecnologías Utilizadas
#### HTML5: Estructura semántica del sitio.

#### CSS3: Estilos personalizados para una experiencia de usuario (UX) limpia.

#### JavaScript (ES6+): Lógica de programación y cálculos financieros.

#### jQuery (3.6.4): Manejo de eventos y simplificación de la manipulación del DOM.
